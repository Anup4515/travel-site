# Golden Triangle Landmark Images

This directory contains background images for the hero section slideshow on the AureoTravels homepage.

## Required Images

Place the following images in this directory:

1. **taj-mahal.jpg** - Taj Mahal in Agra
2. **red-fort.jpg** - Red Fort in Delhi
3. **amber-fort.jpg** - Amber Fort in Jaipur
4. **hawa-mahal.jpg** - Hawa Mahal (Palace of Winds) in Jaipur
5. **india-gate.jpg** - India Gate in Delhi
6. **qutub-minar.jpg** - Qutub Minar in Delhi

## Image Requirements

- **Format**: JPG (recommended for photos)
- **Dimensions**: Minimum 1920x1080 pixels (Full HD)
- **Quality**: High resolution for optimal display
- **Aspect Ratio**: 16:9 recommended
- **File Size**: Optimize for web (aim for under 500KB per image)

## Current Status

The slideshow component is configured to cycle through these images every 5 seconds with smooth fade and zoom transitions.

## Notes

- Images are automatically optimized by Next.js Image component
- The first image (taj-mahal.jpg) is preloaded for faster initial display
- All images have a dark overlay (60% opacity) to ensure text readability
